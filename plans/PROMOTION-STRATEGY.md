# 「我在」小程序推广策略方案

> 状态：方案设计，待确认后执行
> 适用阶段：MVP验证期 → 冷启动增长期
> 核心约束：单人开发、零预算或极低预算、不依赖买量

---

## 一、核心判断：不靠搜索，靠入口

小程序800万+竞品中靠搜一搜获取AI类产品流量，天花板极低（每天几十个搜索流量）。

**正确的获客逻辑：用户不是搜索到「我在」的，是被一个具体的场景问题和信任关系带进来的。**

所以推广策略围绕一条主线设计：

> **制造一次「不用组织语言、说完就清楚了」的体验 → 用户感受到价值 → 自然分享 → 人传人**

---

## 二、推广蓝图：三层漏斗

```
第一层：入口（告诉用户"这能帮你做什么"）
  ↓
第二层：体验（3分钟内让用户感受到价值）
  ↓
第三层：传播（用户主动替他分享）
```

### 第一层：入口设计（怎么让人第一次点进来）

#### 1. 小程序名字 = 搜索关键词（零成本SEO）

不要叫「我在」——用户不会搜这个词。

推荐备选名（按优先级）：

| 名字 | 用户搜索场景 | 竞争度 |
|------|-------------|--------|
| **思绪助手** | "脑子乱""理清思绪" | 低（仅1-2个同类） |
| **AI倾听** | "找人说说""心里话" | 中 |
| **有话说** | "有些话不知道跟谁说" | 低 |
| **讲出来** | "说出来就舒服了" | 低 |

**取名原则：用户在有需求的时候，会搜什么词？**

名字可以跟小程序内部的品牌名不一样。小程序名叫「思绪助手」，内部显示「我在 × 思绪助手」即可。

#### 2. 微信签名 = 活广告位，零成本，每天被人看到

把你的微信签名改成：

> 「做了个小东西：脑子乱的时候说出来，它帮你理清。要用的私信我」

这不是发广告，是让朋友圈的人知道你做了这个。**每天被几百人看到。**

#### 3. 发一条真实的朋友圈，不是广告

不要写：「推荐一个好用的AI工具」

写：「这周试用自己写的一个小东西。下班后把脑子里乱七八糟的事说出来，它帮我理出了真正烦的是什么——有些事以前只有自己知道，说出来之后反而清楚了。有需要的私信我。」

**真实体验比任何广告文案都有效。**

#### 4. 一对一私信种子用户

不做群发。从你的177人团队和前同事中，找出5-10个符合以下特征的人：

- 工作压力大、经常说"脑子乱"
- 最近在做选择（职业/家庭/生活）
- 信任你的判断

一对一说：「我自己做了一个小东西。有件事在脑子里翻来覆去的时候，不用组织语言，直接跟它说。它会记住你的情况，越说越懂你。你愿意帮我用用看吗？」

#### 5. 小红书内容引流（中长期）

不发"产品推荐"，发**真实体验帖**：

- 「有件事在脑子里反复想了一周，说出来之后发现真正纠结的根本不是我以为的那件事」
- 「跟AI聊了三天，它比我妈还了解我在纠结什么」
- 「有些话不能跟同事说、不想跟家人说，但需要一个地方说出来」

**内容策略：制造共鸣，不制造购买冲动。**

每条内容结尾：「想知道我怎么做的可以私信我」→ 引导到私域。

---

### 第二层：体验设计（3分钟内证明价值）

入口把人带进来了，但ta只会给一次机会。前3轮对话决定留不留。

#### 必须做好的几个设计：

**1. 第一句话决定存亡**

不要：

> 你好，我是你在，你的AI数字分身，我可以帮你……

要做：

> 随便说一件最近在想的事，不用组织语言，说完就行。

**2. 回复结构固定**

用户发完语音/文字后，「我在」回复结构：

```
我听懂的是：
你真正在意的可能是：
事实 vs 担忧：
现在可以做的一个最小动作：
```

**不需要从第一句就开始「了解用户」。第一轮就让用户觉得「它听懂了」。**

**3. 3分钟价值证明**

如果用户在3轮对话内感觉「它确实在听而且理清了」，ta就会信任。

如果只是普通AI聊天，3轮后用户就走了。

#### 不需要一开始就做好的：

- X登录注册
- X长期记忆展示
- X复杂技能路由
- X多轮Agent Loop

**MVP推广期只做一件事：让用户说一件事，你理清它，用户觉得有用。**

---

### 第三层：传播设计（用户为什么帮他分享）

「我在」的核心矛盾：对话是私密的，不能分享内容。

但是可以分享以下东西：

#### 1. 可分享的体验卡（每轮对话自动生成）

用户完成一次思绪整理后，显示：

> 「这一轮它帮你理清了：你真正在意的是 XX」
> 「分享给朋友」→ 生成一张卡片，只写「我把这件事说清楚了」+ 一个随机emoji

**不包含具体内容，但朋友看到后会说"这是什么？"**

#### 2. 思绪云（不含具体内容的可视化）

用户完成3轮后，可以生成一张「思绪云图」——关键词频率，但不是具体内容。

例如出现最多的词是：工作、选择、焦虑、家庭

用户觉得「这就是我这周在想的」，愿意发朋友圈。

**这张图本身就是一个有效的广告。**

#### 3. 邀请朋友 = 获得新能力

- 邀请1个朋友 → 解锁「每周报告」
- 邀请3个朋友 → 解锁「无限历史回顾」
- 邀请5个朋友 → 解锁「深度分析」

**核心：不伤害隐私，但给分享动力。**

#### 4. "它说了一句让我记住的话"

用户可以在对话中选择一句ta想记住的话，一键生成卡片分享。

例如：

> 「它说：你不是在逃避，你是在等一个更确定的信号。」

这句话是用户在对话框里手动选一句分享的——**保护隐私，同时制造传播。**

## 三、冷启动节奏（30天）

### 第1周：基础设置 + 种子用户

- [ ] 确定小程序名称（如「思绪助手」）
- [ ] 修改微信签名
- [ ] 发一条朋友圈
- [ ] 私信5-10个种子用户

### 第2周：收集反馈 + 完善体验

- [ ] 每天问3个种子用户：哪里有用？哪里不对？
- [ ] 根据反馈改进回复结构和体验
- [ ] 记录3个「它记住我了」的真实案例

### 第3周：内容 + 分享功能上线

- [ ] 发1-2条小红书
- [ ] 上线「分享体验卡」功能
- [ ] 上线「邀请奖励」

### 第4周：复盘 + 放大

- [ ] 统计：多少种子用户留下来了？多久回访一次？
- [ ] 收集愿意公开的案例（脱敏后）
- [ ] 决定是否加大力度（换小程序名、投小金额广告、发更多内容）

## 四、不做什么（反模式）

| 不要做 | 为什么 |
|--------|--------|
| 投微信广告 | 预算门槛1万+，获客成本高，不适合冷启动 |
| 做公众号涨粉再转化 | 太慢，用户被公众号内容分散注意力 |
| 买KOL推广 | 用户看到"广告"就不信任了 |
| 设计复杂分享裂变 | 诱导分享被微信限制，损坏产品形象 |
| 一开始就做独立APP | 获客成本太高，小程序验证后再考虑 |

## 五、成功指标（30天）

| 指标 | 底线目标 | 理想目标 |
|------|---------|---------|
| 种子用户 | 30人 | 50人+ |
| 7天留存 | 10人（33%） | 20人（40%） |
| 主动分享 | 3人 | 10人 |
| 月活跃对话 | 50轮 | 200轮+ |
| "它记住我了"反馈 | 5条 | 15条 |

---

## 六、一句话总结

> **不要推广"AI分身"，推广"把一件事说出来就清楚了"。**
> 入口是朋友圈和一对一私信，
> 价值在第一次对话的3分钟内证明，
> 传播靠分享不泄露隐私的体验卡片。
